## <img src="https://media.giphy.com/media/hvRJCLFzcasrR4ia7z/giphy.gif" width="28" /> Engineering Focus & About Me

I'm a **B.Tech CSE student @ IIIT Kota** specializing in designing and deploying robust AI architectures and high-performance backend systems. I focus on building production-ready pipelines that prioritize sub-second latency, optimal time complexity, and scalability. 

**Core Competencies:**

- 🧠 **AI & Machine Learning** — Architecting RAG pipelines, Agentic Workflows (LangGraph), vector embeddings (LanceDB), and integrating LLMs into robust backend systems.
- ⚡ **Backend Architecture** — Building event-driven microservices, high-throughput APIs (FastAPI), Redis Streams, and WebSocket-driven real-time communication.
- ☁️ **Infrastructure & DevOps** — Containerization (Docker), automated CI/CD workflows via GitHub Actions, and scalable cloud deployments on Linux environments.
- 🎯 **Algorithmic Problem Solving** — 300+ LeetCode, 1400+ Codeforces Specialist. Obsessed with writing optimal, time-complex algorithms and graph-native structures.
